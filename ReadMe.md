# Projeto de Autenticação (Login & Cadastro)

Este projeto demonstra minhas habilidades iniciais em **Desenvolvimento Web**, focando na construção de interfaces e na lógica de interação com o usuário. Ele é o ponto de partida de um sistema que receberá atualizações constantes.

## Sobre o Projeto

O objetivo principal desta primeira versão é apresentar a criação de telas de **Login e Cadastro** e a manipulação de dados básicos. 

> [!IMPORTANT]
> Este é um projeto de nível **básico**, fruto dos meus estudos atuais. O código foi desenvolvido de forma simples e didática, priorizando o aprendizado dos fundamentos antes de partir para arquiteturas mais robustas.

### O que foi implementado:
* **Interface de Usuário:** Criação de formulários estruturados.
* **Interação com JavaScript:** Captura de eventos de clique e envio de formulário.
* **Feedback via Console:** Os dados inseridos são processados e exibidos no console do navegador para validar a lógica de programação.
* **Design Assistido:** A paleta de cores e a harmonia visual contaram com o auxílio de **IA** para garantir uma estética moderna e limpa.

---

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização responsiva.
* **JavaScript (Vanilla):** Lógica de interação e manipulação do DOM.
